Introduction

Cette documentation technique explique les fondamentaux de HTML. Elle est conçue pour respecter les critères FreeCodeCamp.

HTML (HyperText Markup Language) est le langage standard utilisé pour structurer et présenter le contenu sur le web.

  • Langage de balisage
  • Structure des pages web
  • Base pour CSS et JavaScript
<!DOCTYPE html>
<html>
  <head></head>
  <body></body>
</html>
Bases HTML

HTML utilise des balises pour définir le contenu. Chaque balise a un nom et peut contenir du texte ou d'autres balises.

Les éléments de base :

  • <html> définit le document HTML
  • <head> contient les métadonnées
  • <body> contient le contenu visible
<html>
  <head>
    <title>Titre</title>
  </head>
  <body>Contenu</body>
</html>
Structure HTML

Un document HTML standard contient les sections suivantes :

Chaque section joue un rôle précis dans l’organisation du code HTML et assure une bonne lisibilité.

  • Doctype
  • <html>
  • <head>
  • <body>
<!DOCTYPE html>
<html lang="fr">
  <head>
    <meta charset="UTF-8" />
    <title>Ma page</title>
  </head>
  <body>
    <h1>Bonjour</h1>
  </body>
</html>
Elements HTML

Les éléments HTML définissent la structure du contenu :

  • <h1> à <h6> : titres
  • <p> : paragraphes
  • <a> : liens
  • <img> : images
  • <ul> et <li> : listes
<h1>Titre principal</h1>
<p>Paragraphe de texte.</p>
<a href="#">Un lien</a>
Bonnes pratiques

Quelques conseils pour écrire du HTML propre :

  • Utiliser une indentation cohérente
  • Fermer correctement les balises
  • Utiliser des attributs valides
  • Structurer le document avec des sections sémantiques
  • Tester le code sur différents navigateurs
<header>Titre</header>
<main>Contenu</main>
<footer>Pied de page</footer>
